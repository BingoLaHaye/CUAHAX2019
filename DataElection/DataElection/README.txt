Data is obtained from the following websites:

https://electionresults.dcboe.org/
https://www.dcboe.org/Elections/Election-Results-Archives

Check this websites as they provide detailed information about the
data.

This folder is organized by election years. For each year, we have
downloaded the data as provided by the above websites. Again, we urge
you to go to these websites and see the data description. It is also
necessary to visit these websites to match the final names with the
election that took place (especially for years prior to 2010).

Data from recent elections is available in .xls format (2010--2018),
.csv (2011--2018), and .pdf (2010--2018). Data for prior years is
available in .pdf, .txt, or .html files.
